# Calculadora de Dobras Cutâneas

Ferramenta web para ensino da disciplina **Medidas e Avaliação em Educação Física**.

Calcula:

- Soma das dobras cutâneas
- Densidade corporal (Jackson & Pollock)
- Percentual de gordura (equação de Siri)

Com decomposição passo a passo dos cálculos.
